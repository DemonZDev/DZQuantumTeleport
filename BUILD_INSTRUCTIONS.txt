🚀 DEPLOYMENT STEPS

1. Build the plugin:
   ./gradlew clean shadowJar

2. Locate the JAR:
   build/libs/DZQuantumTeleport-1.0.0.jar

3. Install dependencies on your PaperMC 1.21.1 server:
   - DZEconomy plugin
   - LuckPerms plugin
   - (Optional) PlaceholderAPI, WorldEdit, WorldGuard

4. Copy JAR to server:
   server/plugins/DZQuantumTeleport-1.0.0.jar

5. Start/restart server

6. Configure:
   plugins/DZQuantumTeleport/config.yml
   plugins/DZQuantumTeleport/ranks.yml
   plugins/DZQuantumTeleport/messages.yml
